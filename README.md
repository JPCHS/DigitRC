## 使用依赖

- Python 3.9 - 3.10

- 依赖库：

```properties
numpy==1.26.2
opencv-python==4.8.0.74
Pillow==9.1.0
tk==0.1.0
torch==2.0.1
torchvision==0.15.2
```

## 如何开始

* 解压压缩包
* 启动 `DigitRC.exe`
* 你可以在出现的窗口中写入数字，完成后将会自动识别
* 请保证你书写的各位数字之间没有笔画相连
* 出现识别的数字之后你仍然可以继续书写，程序会自动清屏
* 你也可以在 `DigitRC/_internal/source` 中找到源程序文件 `DigitRC.pyw` 并运行
